Events are generated by POSTing to features.
Events can be websocketed - still whole JSON entities.

================================

Endpoints:
	/features 					- GET
	/features/{feature} - GET, POST, PUT
	/events 						- GET

Query Params:
  lat, lon, range
  offset, count
  after, before
